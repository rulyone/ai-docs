# A day.

A building.

Floor by floor.

*Form follows function.*

---

## Three loads.

| Load        | Bears        | Fails    |
| ----------- | ------------ | -------- |
| Compression | Work         | Burnout  |
| Tension     | Attention    | Drift    |
| Shear       | Transitions  | Friction |

Design the joints.

---

## Compose. Reduce. Close.

- Compose, don't consume.
- Pour the morning slow.
- Reduce to load-bearing.
- Empty rooms breathe.
- Close deliberately.

*The blueprint is yours.*
